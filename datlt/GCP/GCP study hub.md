# Foundational concept

## Cloud service 

IaaS: VM on cloud. Example: EC2, compute engine, Azure VMs
PaaS: Don't care about infra, focus on code. Example: GKE, Pub/Sub, Amazon EMR,...
SaaS: Access software application via internet. Example: google sheets, msoffice,..

## Data life cycle

**Ingest**: Cloud pub/sub, cloud dataflow, cloud iot core, logging, cloud storage
	Transfer services: transfer appliance, storage transfer service, bigquery data transfer service

**Store**: cloud storage, bigquery, cloud sql, cloud bigtable, cloud memorystore, cloud spanner, filestore 

**Process**: dataproc, bigquery, dataflow, cloud data fusion, data loss prevention api, dataprep, cloud functions, cloud composer

**Analyze and Extract Value**: bigquery, looker, dataproc, dataflow, dataprep, cloud bigtable, vertex ai

**Governance**: iam, dataflex, cloud logging, data catalog, data loss prevention api, vpc, org policy service, key management service
